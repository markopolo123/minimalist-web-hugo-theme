This is a Hugo theme inspired by [The Monospace Web](https://owickstrom.github.io/the-monospace-web/). It embraces a monospace-first, minimalist aesthetic.

## Features

- Monospace typography throughout
- Character-based responsive design
- Automatic dark mode support
- ASCII art and diagram support
- Minimalist, distraction-free reading
- Fast and lightweight
